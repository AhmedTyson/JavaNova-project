## <img src="https://api.iconify.design/lucide:info.svg?color=%238A2BE2" width="20" height="20" align="center" /> Overview

JavaNova Academy is a fully realized education platform landing page for a Java programming school. The site was generated through a detailed design prompt using DeepSeek-R1 and Perplexity Pro AI models, then refined and deployed as a GitHub Pages site. It features a modern space-themed aesthetic, structured course sections, instructor profiles, and a call-to-action enrollment flow. The project exists in two versioned iterations (`v1.5` and `v2.3`), reflecting an iterative AI-assisted design process.
